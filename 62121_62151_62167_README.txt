Тема на проект: 9.3 Генериране на разписание
Подтема 1: Цялостно разписание на предстоящи/презентирали
Подтема 2: Лично разписание – според проявен ‚интерес‘
Подтема 3: Разпечатване на разписанието

Забележка: Това не е финална версия на проекта. 

Този архив съдържа следните компоненти на проекта:
1) HTML файлове за визуализация на потребителския интерфейс
2) Директория script, съдържаща JavaScript файлове за осъществяване на връзката със сървърната част
3) Директория style, съдържаща CSS файлове за стилизиране на потребителския интерфейс
4) Директория php, съдържаща PHP и MySQL файлове за реализиране на сървърната част

За финална версия на проекта се планира да се допълни следното:
1) Документация на проекта
2) Функционалност да се разпечатва разписание като .pdf и други формати
3) Инструкции за компилация на проекта в този README файл
